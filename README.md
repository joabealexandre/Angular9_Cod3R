## Como rodar o projeto

Navergar para o diretório ./backend e executar os comandos
* npm install

    e

* npm start

Navergar para o diretório ./frontend/ e executar os comandos

* npm install

    e

* ng s

Abrir a URL http://localhost:4200/ no navegador
